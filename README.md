# Смотрелка — интерактивный прототип

HTML-прототип нового раздела «Смотрелка» с айфоновским мокапом. Показывает основную механику продукта: вход из нижнего бара → выбор класса/предмета/темы → урок (ролик + свайп) → проверочное задание → результаты.

## Как открыть

Рекомендуется запустить локальный сервер (чтобы корректно проигрывались видео):

```bash
cd "Прототип смотрелки 2.0"
python3 -m http.server 4599
# открыть http://localhost:4599
```

Можно просто открыть `index.html` двойным кликом в Chrome.

**Про звук и автозапуск.** Видео стартует без звука (иначе браузер не даёт автозапуск). Звук включается **кнопкой динамика** справа сверху или по тапу на подсказку «Нажмите для звука». Если браузер заблокировал автозапуск, показывается кнопка **«Нажмите, чтобы включить»**.

**Взаимодействие с роликом.** По умолчанию кнопки (лайк / дизлайк / поделиться / проблема) скрыты. **Тап по видео** ставит ролик на паузу и показывает кнопки; повторный тап — продолжает и скрывает. **Свайп вверх** — переход к заданию. Подсказка «Смахни вверх» появляется после первого полного просмотра ролика.

**Ошибка в задании.** Если в квизе выбран неверный ответ — показывается мягкое сообщение и предложение пересмотреть ролик. После возврата к ролику и свайпа вверх задание начинается заново.

## Выбор персонажа и прямые ссылки

При открытии показывается технический экран **«Кто проведёт урок?»** — выбор из 4 персонажей. Аватар каждого — живой кадр (лицо) из его ролика. Выбранный персонаж определяет, чьи ролики играют в «Смотрелке». Сменить персонажа можно чипом-аватаром в правом верхнем углу главной.

| Персонаж | Описание | slug |
|---|---|---|
| **Кира** | Главная по идеям (девушка) | `kira` |
| **Тим** | Старший брат Тёмки (тигр, синяя худи) | `temka1` |
| **Тёмка** | Мастер приключений (тигр-маскот, клетчатая рубашка) | `temka2` |
| **Макс** | Энергичный блогер (парень) | `maks` |

> Слаги в URL остались `temka1`/`temka2` (стабильные ссылки для деплоя): `temka1` = **Тим**, `temka2` = **Тёмка**.

**Прямые ссылки (через слэш):** `http://localhost:4599/kira`, `/temka1`, `/temka2`, `/maks` — открывают прототип сразу с этим персонажем, минуя экран выбора. (Также поддерживаются `?c=<slug>` и `#c=<slug>`.)

**Ролики по персонажам:** каждый персонаж проигрывает свои ролики из `assets/videos/<slug>/lessonN.mp4` (флаг `PER_CHAR_VIDEOS = true` в `index.html`). Чтобы обновить/добавить ролики — заменить файлы в соответствующей папке.

## Флоу (что показать)

1. **Экран выбора персонажа** → выбор героя → **Главная приложения** → тап центральной кнопки **«Смотрелка»** в нижнем баре
2. **Главная Смотрелки** «Предмет и тема» — фильтры «Класс»/«Предмет», список тем, кнопка «Смотреть все подряд» / «Продолжить»
3. **Выбор класса/предмета** — нижний шит с радиокнопками и «Выбрать» (смена на не-«Математика, 1 класс» показывает пустое состояние)
4. **Экран темы** «На сколько больше? На сколько меньше?» — 5 уроков, прогресс, «Продолжить»
5. **Экран ролика** — реальное видео (зациклено), реакции справа, прогресс-бар; через ~7 сек появляется подсказка **«Смахни вверх»**
6. **Свайп вверх** (на десктопе — потянуть мышью вверх / колесо вверх / клик по подсказке) → **игра**
7. **Задание** — одна из механик, отсчёт «3-2-1», прогресс «N из 3»
8. **Результаты** — промежуточный («Продолжить» к след. уроку) или итоговый («Тема пройдена!»)

## Механики по урокам (контент из ТЗ)

| Урок | Название | Механика |
|---|---|---|
| 1 | На сколько больше? | Квиз (3 вопроса, с картинками) |
| 2 | На сколько меньше? | Соедини пары (текст ↔ текст) |
| 3 | Условные обозначения | Квиз, в т.ч. картинки-ответы |
| 4 | Как заполнить таблицу по рисунку | Соедини пары (текст ↔ значок) |
| 5 | Как сравнить значки в таблице | Квиз с картинками-таблицами |

## Структура

- `index.html` — весь прототип (разметка, стили, логика; один файл)
- `assets/videos/lesson1..5.mp4` — 5 роликов темы
- `assets/img/image1..14.png` — картинки заданий (извлечены из контента)
- `.claude/launch.json` — конфиг локального сервера

## Заметки

- Прогресс хранится в памяти вкладки и сбрасывается при перезагрузке (удобно для повторных демо).
- Дизайн снят с Figma-страницы «Смотрелка claudeReady» (Лекта — Master File); экраны заданий/результатов дорисованы в том же визуальном языке.
- Коллекционные предметы (награды) в этот прототип не включены — по договорённости показан только основной флоу.
